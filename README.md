## 📌 Project Overview

This project is a complete **Data Science and Data Analytics solution** built using Python. It analyzes the **Netflix Movies and TV Shows dataset** to extract meaningful insights and build an interactive dashboard.

The project covers:
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Interactive Dashboard using Streamlit

It demonstrates an end-to-end **Data Science workflow from raw data to insights and interactive visualization**.

Netflix Movies and TV Shows Dataset:
- Source: Kaggle  
- Contains information about movies and TV shows including:
  - Title
  - Type (Movie / TV Show)
  - Director
  - Cast
  - Country
  - Release Year
  - Rating
  - Duration
  - Genres

## 📌 Features

### 🔹 Data Cleaning
- Handling missing values
- Removing duplicates
- Formatting date and duration columns

### 🔹 Exploratory Data Analysis (EDA)
- Content distribution analysis
- Country-wise content analysis
- Ratings and genres insights
- Trend analysis over years

### 🔹 Data Visualization
- Bar charts
- Line plots
- Heatmaps
- Distribution plots

### 🔹 Interactive Dashboard
- Filter by Movie / TV Show
- KPI metrics (Total Titles, Movies, TV Shows)
- Dynamic visualizations
- Data preview table

## 📊 Key Insights

- Netflix has more **Movies than TV Shows**
- USA is the leading content producer
- Content production increased significantly after 2015
- TV-MA is the most common rating
- International content is rapidly growing
